简述

因工作需要，开始接触并学习 MQTT 协议及相关的技术，故开此系列记录个人学习成果。完善此系列的过程中，力求知其然，知其所以然。

系列内容

亦或称学习目标。当前期望学习如下内容：

  • MQTT 协议
  • EMQ X 消息服务器
  • MQTT 客户端实现库
    • 浏览器和 Node.js 库：MQTT.js
    • Java 库：待定

学习路线

  1. 了解 MQTT 的历史 ——— 了解工具存在的意义，才能用得得心应手

  2. 学习 MQTT 协议的大致框架，包括：

    • 协议基本格式
    • 包类型

  3. 学习 MQTT 的作用流程

  4. 学习 EMQ X 服务器（穿插学习不同包的特殊包格式）

    • 优缺点
    • 搭建、配置
    • 配合客户端使用

  5. 学习 MQTT 实现库

    • MQTT.js

对于协议规范部分会根据需要照搬规范原文且翻译总结。
